1. Treat only `ProcessLookupError` and `OSError(errno.ESRCH)` as vanished
   `/proc` entries; all unrelated I/O, permission, parsing, and monitoring
   failures continue to propagate.
2. Require the supervisor/root PID to contribute a positive RSS sample.
3. Preserve the original approval packet and economic campaign artifacts.
   Resume gates bind the reviewed implementation at repository authority
   `7e7c7b47a7693b7923097f01c22b4b7287b6d971` without regenerating economic
   authority.
4. Reuse only hash-reconciled completed jobs and submit only missing jobs.
5. Preserve the original campaign deadline and audit reserve. The later human
   request to remove the stop required no action because the campaign had
   already reached terminal completion; the frozen runtime contract was not
   altered retroactively.
6. Keep large event/scoring tapes local and hash-manifested. The Drive handoff
   contains only compact, reviewed evidence.
